## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/phoenixdev100/job-portal.git
cd job-portal
```

2. **Install dependencies**
```bash
npm install
```

3. **Set up environment variables**
   - Copy `.env.example` to `.env`
   - Update the values with your Sanity credentials:
     ```
     REACT_APP_PROJECT_ID=your_sanity_project_id
     REACT_APP_DATASET=production
     REACT_APP_BASE_URL=your_sanity_api_url
     REACT_APP_BEARER_TOKEN=your_sanity_token
     ```

4. **Start the development server**
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)
